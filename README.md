# Note CLI

Note CLI is a small program intended for taking note in command line.

## Usage

1. Build the project.
2. Run the program in your terminal.

## Commands

1. Help

```shell
note_cli help
Note CLI - Take a short note in command line.
Usage:
    list        List of notes
    create      Create new note
    read        Read note
    delete      Delete note
```

2. List

```shell
note_cli list
List of notes:
1: test
```
